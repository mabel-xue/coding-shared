# What's Improvement

## 🔧 ts中继承的最佳实践 | ts/最佳实践

抽象类: abstract 修饰， 里面可以没有抽象方法。但有抽象方法(abstract method)的类必须声明为抽象类(abstract class)

```js
/**
 * Animal 是一个抽象类，里面含有一个eat()抽象方法
 */
abstract class Animal{
    public name:string;
    constructor(name:string){
        this.name=name;
    }
    //抽象方法 ，不包含具体实现，要求子类中必须实现此方法
    abstract eat():any;
    //非抽象方法，无需要求子类实现、重写
    run(){
        console.log('非抽象方法，不要子类实现、重写');
    }
}
```
