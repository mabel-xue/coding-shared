## lodash 常用方法 | js/库

### Array

```js
_.fill(array, value, [start=0], [end=array.length])

_.head(array) -> array[0]

_.indexOf(array, value, [fromIndex=0]) -> array.indexOf(value)

_.join(array, [separator=',']) -> array.join(separator)
```

好处：有类型检查 且 对错误类型有相应处理
